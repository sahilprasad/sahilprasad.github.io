Me as a student.

Courses:

  • CS17/18 - Integrated Introduction to Computer Science
    • This was the intermediate introductory sequence offered at Brown for potential computer science concentrators. We used two functional languages (Racket and OCaml) to learn about basic algorithms, data structures, runtime complexity, etc. Some examples of projects done for the class are: a Gomoku remake with an A.I. agent, a search engine, a Racket interpreter made in OCaml, and a server-client system mimicking the controls of Git.
  • CS22 - Discrete Math and Probability
    • Introductory class that focused on the mathematical theory behind many computer science concepts, although not restrained to them. Topics included probability, number, and graph theory.
  • CS33 - Introduction to Systems
    • This class reinforced many of the high-level concepts I had learned thus far in my education with low-level explanations. Using C, we implemented projects such as a dynamic memory allocator, buffer overflow hack, and several more.
  • CS1270 - Database Management Concepts
    • CS1270 provided a theoretical backing to my experience with databases, covering topics such as query optimization, B+ trees for index building, the various normal forms, and much more.
  • CS1300 - User Interface Design
    • This was the first class I had ever taken focused on the more user-facing aspects of software engineering, and I learned a lot about how best to design applications to ensure appropriate usage and a satisfactory experience. The final project for the class required us to get into groups and produce an interface from the ground up. My group's project, Point, is a redesign of a typical map interface that is optimized for pedestrians, cyclists, and runners.

  • CS1320 - Creating Modern Web Apps (Currently enrolled)
